# Testing Project — HVT Vignettes

This repository contains R Markdown vignettes and their rendered HTML outputs for the **HVT (Hierarchical Voronoi Tessellations)** package.

## Contents

| File | Description |
|------|-------------|
| `Dynamic_Forecasting_macroeconomic_data.Rmd` | Vignette demonstrating dynamic forecasting of macroeconomic time series data using HVT with Monte Carlo Simulations of Markov Chain (MSM). |
| `Dynamic_Forecasting_macroeconomic_data.html` | Rendered HTML output of the dynamic forecasting vignette. |
| `Experimentation_of_hyperparameters_in_msm.Rmd` | Vignette for hyperparameter experimentation and champion model selection in MSM dynamic forecasting. |
| `Experimentation_of_hyperparameters_in_msm.html` | Rendered HTML output of the hyperparameter experimentation vignette. |

## About HVT

The HVT package provides a suite of R functions for constructing topology-preserving maps using Hierarchical Voronoi Tessellations. It supports data compression, dimensionality reduction, tessellation, scoring, temporal analysis, and dynamic forecasting.

For more details, visit the [HVT GitHub Repository](https://github.com/Mu-Sigma/HVT).
